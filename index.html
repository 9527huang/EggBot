Eggbot

Software for the Original EggBot Kit

Software repository for The Original EggBot kit, by Evil Mad Scientist Laboratories. More information at http://www.egg-bot.com/

View on GitHub

This repository includes the extensions for Inkscape, firmware for the EiBotBoard (AKA "EBB", or EggBotBoard), and example plot files. All code is designed to work on Mac, Windows and Linux, and on every model of EggBot.

For download and installation instructions, please start at our main documentation wiki.

Extended documentation available at: http://wiki.evilmadscientist.com/eggbot
